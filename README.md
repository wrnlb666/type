# type
Runtime type system for C. 


## STILL IN PROGRESS!!!


## current behavior: 

* n: nil
* i: int
* u: uint
* f: float
* s: string
* a: array/struct
* l: list 
* d: dict
* (): array with values
* []: list with values 

e.g.: `(u[si])` meaning an `array` containing a `uint` and a `list` containing a `string` and an `int`
